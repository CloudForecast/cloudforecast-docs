## What is AWS MAP?

The AWS Migration Acceleration Program (MAP) is an AWS program that provides credits to help offset cloud costs during a migration engagement. When your organization qualifies for MAP, AWS allocates a pool of credits that apply to eligible services over the duration of your migration contract. MAP credits can meaningfully reduce your AWS bill during migration — but only if you are actively tracking how they are being consumed relative to your spend targets.

## What the MAP Tracker does

The CloudForecast MAP 2.0 Tracker gives you a dedicated dashboard to monitor your MAP engagement from start to finish. It connects your MAP contract details to your actual AWS spending data, so you can see at a glance:

- How much of your allocated credit has been used
- How your migration spend compares to your target
- Whether you are on track to fully utilize your credits before the engagement ends

If you are managing a MAP engagement on behalf of a client, the tracker lets you include both your team and your client's stakeholders on the notifications, keeping everyone aligned without extra manual reporting.

## What you configure

Each MAP Tracker is set up with the following information:

| Field | Description |
|-------|-------------|
| **Report Name** | A name for this migration engagement (e.g., "Acme Corp Migration") |
| **MPE ID** | The MAP Program Enrollment ID from your MAP agreement with AWS |
| **Contract Start Date** | When the MAP engagement officially began |
| **Migration Spend Target** | The total AWS spend target committed to in the MAP agreement |
| **Total Credits Allocated** | The total dollar amount of MAP credits AWS has allocated |
| **Service Codes** | The AWS service codes eligible for MAP, DB&A, or SAP/Oracle credit categories |
| **Sub Accounts** | Optionally restrict tracking to specific AWS accounts within your organization |
| **Client Emails** | Email addresses for client stakeholders who should receive MAP reports |
| **AWS Rep Emails** | Optionally include your AWS account team on notifications |

Service codes are organized into three categories that MAP defines separately: MAP 2.0 service codes, DB&A (Distributed, Big Data & Analytics) service codes, and SAP and Oracle Applications service codes. You can configure each category independently.

## The MAP dashboard

The MAP dashboard shows your credit usage and spend progress over time. It gives you visibility into how credits are being applied against your eligible AWS spending, and whether you are on pace to meet the migration spend target required to unlock the full credit allocation.

This view is useful both for internal tracking and for sharing progress with clients and AWS account teams at regular check-ins.

## Who this is for

The MAP Tracker is built for AWS partners and enterprise teams that are actively running a MAP-funded migration engagement. If your organization has received MAP credits from AWS and wants to track utilization against your program targets without building custom reporting, the MAP Tracker handles that automatically using your connected AWS billing data.
